# SpecLock Design Partner Pipeline

20 developers identified via GitHub research (April 2026) who actively use AI coding tools (Claude Code, Cursor, Codex, Cline, Aider, Copilot CLI) and have publicly complained about AI agents ignoring their rules / CLAUDE.md / AGENTS.md / .cursorrules / .clinerules / CONVENTIONS files.

Research method: web-search of GitHub issues matching phrases like "CLAUDE.md ignored", "cursorrules not working", "clinerules completely ignored", "AGENTS.md ignored", then per-author profile lookup. Candidates with 0 public repos, bots, or obviously inactive accounts were dropped.

Note on emails: GitHub rarely exposes a public `.email` field for individual devs anymore. Where a profile didn't expose one, the most reliable contact path is (a) replying on the issue itself (tags the user and is on-topic), (b) the author's personal site / blog / company page, or (c) their linked X/Twitter. The "Email/Contact" column records the best available path. Contacting via the original issue thread is usually the strongest opener because the pitch is already context-matched to what they complained about.
